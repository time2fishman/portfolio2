Face Detection App

This is a full stack project! The front end is built with React. The backend/API is built with Node.js and Knex. The Database was built using PostgreSQL. All three projects have been deployed and are live on Render. You can register a new user, sign in and out, submit an image URL and the Clarifai API will find the face(s) in the image! The app also keeps track of how many images have been submitted by each user. Passwords are hashed and PAT is not displayed to aid in user and host security. This app is automatically spun down with inactivity so you may experience a delay upon first sign in.
